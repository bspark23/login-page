# Authentication System

A responsive authentication system with login/signup functionality and dashboard.

## Features

- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **User Authentication**: Complete login and signup system with validation
- **Local Storage**: User session management with localStorage
- **Professional UI**: Modern design with animations and micro-interactions
- **Dashboard**: User-specific dashboard with course cards
- **Form Validation**: Email validation, password requirements, and error handling

## Technologies Used

- HTML5
- CSS3 (with animations and responsive design)
- Vanilla JavaScript
- Local Storage for session management

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Create an account or use the demo functionality
